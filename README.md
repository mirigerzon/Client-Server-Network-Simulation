# Client-Server Network Simulation

A network simulation project demonstrating client-server communication principles.  
The system allows clients to connect to a server, send messages, and display them, simulating real-time network interactions.

---

## Features
- Client-server communication simulation  
- Sending and receiving messages between client and server  
- Real-time display of messages  
- Demonstrates basic network protocols and structure  

---

## Technologies
- HTML  
- CSS  
- SCSS  
- JavaScript  

---

## Purpose
This project is designed to demonstrate the fundamentals of client-server architecture and network communication.  
It provides a hands-on way to understand how data is sent, received, and displayed between clients and servers.
